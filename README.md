This is a Chrome extension starter project built with Next.js.

## Getting Started

1. Clone the repository
2. Run `npm install`
3. Run `npm run build`
4. Open Chrome and navigate to `chrome://extensions/`
5. Click "Load unpacked" and select the `out` directory
6. Enjoy!
